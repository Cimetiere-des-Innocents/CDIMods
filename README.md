# CDI Mods

CDI系列Mod的总仓库。

目前包含的Mod如下：

- [CDILib](https://github.com/Cimetiere-des-Innocents/CDILib)
- [Underground](https://github.com/Cimetiere-des-Innocents/Underground-121)
- [CDICore](https://github.com/Cimetiere-des-Innocents/CDICore)

## 构建与运行

由于`generated`目录被排除在版本控制之外，在构建或运行之前务必先运行根项目的`runData` gradle命令，再运行客户端或构建jar。
